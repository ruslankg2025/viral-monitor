# Viral Monitor — Документация системы

> Локальный аналог Piratex.ai. Анализ вирусных видео, генерация хуков и сценариев.
> Stack: FastAPI 8003 · React/Vite 5174 · SQLite · Multi-provider AI

---

## 🔄 ПАЙПЛАЙНЫ

### Пайплайн 1 — Разбор видео по URL (главная фича)

**Что делает пользователь:** вставляет ссылку на любое видео (YouTube Shorts / Instagram Reel / TikTok / VK Клипы)

**Что выдаёт приложение:**
- Структура видео по секундам (хук → основной контент → CTA)
- Почему стало вирусным (3 причины + эмоциональный триггер)
- Сценарий для телесуфлёра (готовый текст)
- 5 вариантов хуков с психологическим разбором и рейтингом
- Описание к рилсу с хэштегами
- Панель улучшения сценария (Усилить зацепку / Сократить / Конкретики / Переписать начало / Упростить / Чат)

**Сервисы и порядок вызовов:**

```
Пользователь вводит URL
        ↓
[Backend] detect_platform_and_id(url)
        ↓ определяет: youtube / instagram / tiktok / vk
        ↓
[Apify] instagram-scraper или tiktok-scraper    ← 1 run (платный после 25/мес)
[yt-dlp] для YouTube — бесплатно
        ↓ метаданные: title, views, likes, duration, thumbnail
        ↓
[AssemblyAI] транскрипция аудио                 ← $0.006 за 60 сек
  (YouTube: сначала пробуем субтитры — бесплатно)
        ↓ текст транскрипции
        ↓
[GPT-4o-mini] суммаризация транскрипции         ← $0.00015 (если >5000 символов)
        ↓ сжатый текст (≤300 слов)
        ↓
[Claude Sonnet] глубокий анализ видео           ← ~$0.015-0.030
        ↓ JSON: hook, structure, why_viral, emotion_trigger
        ↓
[Claude Sonnet] генерация 5 хуков               ← ~$0.010-0.020
        ↓ JSON: 5 × {text, technique, mechanism, rating, timing}
        ↓
[Claude Sonnet] описание к рилсу                ← ~$0.003-0.006
        ↓ description + hashtags[]
        ↓
Результат отображается в UI
```

**Стоимость одного полного разбора:**

| Компонент | Сервис | Стоимость |
|-----------|--------|-----------|
| Парсинг Instagram/TikTok | Apify | ~$0.002/run (в рамках лимита) |
| Парсинг YouTube | yt-dlp | $0 |
| Транскрипция 60 сек видео | AssemblyAI | $0.006 |
| Суммаризация (если нужна) | GPT-4o-mini | $0.0002 |
| AI-анализ структуры | Claude Sonnet | $0.015–$0.030 |
| Генерация хуков | Claude Sonnet | $0.010–$0.020 |
| Описание к рилсу | Claude Sonnet | $0.003–$0.006 |
| **ИТОГО на разбор** | | **$0.034–$0.064** |

### Пайплайн 2 — Улучшение сценария

**Что делает пользователь:** нажимает одну из кнопок или пишет в чат

**Кнопки:** Усилить зацепку · Сократить · Добавить конкретики · Переписать начало · Упростить · Чат

```
Пользователь нажимает кнопку или пишет промпт
        ↓
[Claude Sonnet] improve_script(action, current_text)    ← ~$0.008–$0.015
        ↓ JSON: improved_text, changes_made
        ↓
Текст обновляется в редакторе
```

**Стоимость:** ~$0.008–$0.015 за одно улучшение

### Пайплайн 3 — Мониторинг блогеров (фоновый)

**Что делает пользователь:** добавляет блогера (платформа + username) → нажимает Обновить

**Что выдаёт приложение:**
- Список последних N видео с просмотрами, лайками, ER
- X-Factor = views / median_views (>3x = аутлайер)
- Comment Rate = comments / views × 100
- Лента аутлайеров во вкладке "Тренды"

```
Добавление блогера
        ↓
[Apify] instagram-profile-scraper     ← 1 run (профиль: аватар, followers)
[Apify] instagram-scraper             ← 1 run (видео: views, likes, comments)
  или [yt-dlp]  / [VK API]  / [Apify tiktok]
        ↓ список VideoData
        ↓
[SQLite] upsert видео
        ↓
[Groq / Llama 3.3] категоризация каждого видео   ← $0 (бесплатно)
        ↓ niche, tags, language
        ↓
X-Factor пересчёт (медиана просмотров блогера)
        ↓
APScheduler повторяет каждые 6 часов автоматически
```

**Стоимость на 1 обновление блогера:**

| Компонент | Сервис | Стоимость |
|-----------|--------|-----------|
| Профиль Instagram | Apify | ~$0.001/run |
| Видео Instagram (50 шт.) | Apify | ~$0.002/run |
| Видео YouTube | yt-dlp | $0 |
| Категоризация (50 видео) | Groq | $0 |
| **ИТОГО** | | **~$0.003 (Instagram), $0 (YouTube)** |

### Пайплайн 4 — AI-анализ видео из ленты

**Что делает пользователь:** открывает видео из "Тренды" → нажимает "Анализировать"

```
[Backend] trigger_analysis(video_id)
        ↓ фоновая задача
        ↓
[AssemblyAI / YouTube субтитры] транскрипция    ← $0 или $0.006/мин
        ↓
[Claude Sonnet] analyze_video()                  ← $0.015–$0.030
        ↓
Результат сохраняется, VideoDetail обновляется
```

### Пайплайн 5 — Генерация сценариев (из библиотеки)

**Что делает пользователь:** в разделе "Сценарии" → Новый сценарий → тема / платформа / длина / стиль

